# Public Repository Mirror Policy

## Purpose
This repository is a **read-only public mirror** intended for transparency,
governance, and compliance signaling.

## Allowed Content
- Architecture overviews
- Governance documentation
- Security posture statements
- Compliance intent

## Prohibited Content
- Source code
- Trading strategies
- Execution logic
- Credentials or secrets
- Internal operational details

## Contribution Policy
- Pull requests may be closed without review
- Issues are informational only
- No feature requests accepted

## Change Control
- Changes occur via internal review
- Public repositories mirror approved documentation only

## Enforcement
Violation of this policy may result in:
- Immediate content removal
- Repository archival
- Access revocation
