# Architecture Decision Records (ADR)

This directory contains **Architecture Decision Records (ADRs)** for the
Divergent Trading Systems public documentation mirror.

ADRs document **why** architectural decisions were made, not implementation
details.

## Scope
- High-level system boundaries
- Security and compliance posture
- Governance and operational philosophy

## Out of Scope
- Trading strategies
- Execution logic
- Model design
- Internal tooling

## ADR Lifecycle
- Proposed → Accepted → Superseded
- ADRs are immutable once accepted
- Superseded ADRs remain for audit traceability
