
## ⚠️ Responsible Disclosure Policy

This repository does **not** contain production systems, credentials,
or live execution code.

### Do NOT submit reports for:
- Trading logic
- Execution behavior
- Model performance
- Market interactions
- Private infrastructure

### Accepted Reports
We accept reports **only** for:
- Documentation inaccuracies with security implications
- Policy inconsistencies
- Public-facing configuration errors

### Reporting Process
- Email: security@divergenttradingsystems.com
- Include: description, impact assessment, and public reference if applicable
- Do **not** include proof-of-concept exploits

### Legal Safe Harbor
Good-faith security research conducted under this policy is authorized.
Any activity exceeding this scope is **explicitly unauthorized**.
